# 自動入帳工具下載

## Radiology Billing Entry Tool

配合義大醫院 HIS 系統使用，可自動將記帳單資料輸入至藥品耗材選擇視窗。

### 下載

- [radiology-billing-entry.exe](./radiology-billing-entry.exe) - Windows 版自動入帳程式

### 系統需求

- Windows 10/11
- 義大醫院 HIS 系統存取權限

### 使用方式

1. 在 UDI 管理系統的記帳單列表選擇要入帳的記錄
2. 點擊「匯出 SKU」下載 .sku 檔案
3. 執行入帳程式，選擇下載的 .sku 檔案
4. 程式會自動在 HIS 藥品耗材視窗輸入料號

### 效益

- ⏱️ 節省 90% 入帳時間（每筆從 3 分鐘減至 15 秒）
- ✅ 降低人工輸入錯誤率
- 📋 支援批次處理多筆記錄
